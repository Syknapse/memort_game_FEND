# Memory Game Project

[Try the game here](https://syknapse.github.io/memort_game_FEND/)

A classig memory matching game.

## Dependencies

Font Awesome

## Features

Timer, number of moves, number of matches, rating, reset button, a game over screen with a summary of results.
